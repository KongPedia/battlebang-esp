# `examples/`

Example runtime config and OTA manifest payloads.

- `config.turret_5.json` shows first USB provisioning shape. Replace Wi-Fi, MQTT host, turret id, pose, calibration, and OTA desired build before use.
- `ota-manifest.example.json` shows the public release manifest structure generated by `scripts/turret_fleet/make_release_manifest.py`.

Examples must not contain real Wi-Fi passwords, MQTT passwords, or private broker hostnames.
